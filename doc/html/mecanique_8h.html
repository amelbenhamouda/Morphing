Function Documentation

int inPictureOrButton ( Clic  clic)

Function that checks whether the click is on one of the drawn buttons.

Parameters
clicThe user's click.
Returns
The function returns:
-> returns 0 if it is the button Add constraint point,
-> returns 1 if it's the Hide constraint point button,
-> returns 2 if it is the Start rendering button,
-> returns 3 if it's the <<< button,
-> returns 4 if it's the >>> button,
-> returns 5 if it's the Quit button,
-> returns 6 if we click on the image on the left,
-> returns 7 if we click on the image on the right,
-> returns -1 if it's a bad click.
